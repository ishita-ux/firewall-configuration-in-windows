# firewall-configuration-in-windows
Configured firewall rules to block Telnet port 23 and tested traffic filtering.
# Task 4: Firewall Configuration

## Objective
To configure and test firewall rules to control network traffic.


## Tools Used
- Windows Defender Firewall
- Telnet Client


##  Steps Performed

# Viewed Existing Firewall Rules
Checked inbound and outbound rules.

#Blocked Port 23 (Telnet)
Created inbound rule to block TCP port 23.

# Tested the Rule
Used telnet to verify the port was blocked.

# Verified Blocking
Connection failed, confirming firewall filtering.

# Restored Firewall
Removed the test rule.
